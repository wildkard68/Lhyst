# Lhyst — Compact **Menu** Navigation

**What’s new**
- When space is tight, the tab bar hides and a **Menu** button appears.
- Clicking **Menu** opens a popover listing all sections (Dashboard, Temperatures, Cleaning, Audits, Equipment, Staff).
- Fully keyboard-accessible (↑/↓, Home/End, Enter, Esc) with ARIA roles.

Dark mode toggle and all functionality remain unchanged.
